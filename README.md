# E Commerce Website Performance_Data Analysis 
Interactive dashboard creation - using Tableau
> **Data Objective** ( Purpose of the data ) :

- This is a "E-commerce website logs" data created for helping the data analysts to practice exploratory data analysis and data visualization.
- The dataset has data on when the website was accessed, IP address of the source, Country, language in which website was accessed, amount of sales made by that IP address.
- This dataset supports analysis aimed at understanding user behavior, identifying popular products and regions, evaluating site performance, and optimizing marketing and operational strategies.

> **Dataset Used :** <a href="https://github.com/Pavankumar-Bishnu/Tableau-Project/blob/main/E_Commerce_Website_Logs.csv"> Click Now </a>

- Date of accessing the website
- Country, Language & Platform in which it was accessed
- No. of bytes used & IP address of the person accessing website
- Sales or return amount of that person
- Records ( 1,72,840 )
## **Project Insights & Dataset Overview**

![E-com (Dashboard)](https://github.com/Pavankumar-Bishnu/Tableau_Project_File_1/blob/main/DSBD.png)   


- By membership type, Premium members produce the most sales, followed by Normal. Not Logged In members have zero sales in this sample.
- By payment method, Cash payments generate the highest sales, followed by Credit Card, Debit Card, and Others.
- Session duration averages about 54.2 minutes, with a range from 25 to 83 minutes, indicating long sessions
- Bytes transferred per session average about 1,400, with a median of 579 and a long right tail up to large values, suggesting a few very heavy sessions or file downloads
- Customer age averages about 40.9 years, with most users between 27 (25th percentile) and 39 (50th percentile), showing a predominantly adult, working‑age audience.
- Network protocols are dominated by TCP, with HTTP also common, TCP appears in roughly 63% of rows, HTTP in about 35%, with UDP and ICMP very small shares
- Gender is split into Male and Female only, with a slight female majority (about 59% female vs 41% male).
- Countries are heavily international, with top contributors including Italy, United States, Canada, China, Japan, Russian Federation, Switzerland, India, Poland, and the UK in the earlier slice.
- Languages are dominated by Chinese, followed by Spanish, Russian, French, and English, implying that non‑English markets are the main revenue drivers in this dataset.
  
- By access channel (device/browser), the highest sales come from:
  - Android App (15.87 million)
  - Chrome (11.58 million)
  - Others (10.99 million)
  - Mozilla Firefox (10.83 million)
  - IOS App (8.93 million)
    
- **Top 5 dates by sales**
  - October 6, 2020 (5.63 million)
  - October 31, 2020 (4.49 million)
  - October 29, 2020 (3.87 million)
  - October 4, 2020 (3.70 million)
  - October 10, 2020 (3.63 million)
    
- **Top 5 dates by returned amounts**
  - October 6, 2020 (995K thousand)
  - October 23, 2020 (835K thousand)
  - October 4, 2020 (829K thousand)
  - October 25, 2020 (743K thousand)
  - October 13, 2020 (700K thousand)

## **Final Conclusion:**
- E-commerce website logs dataset is that the platform is experiencing strong user engagement, with clear trends in geographic concentration, product interest, and activity timing.
- The dataset from e-commerce website logs provides several important observations and key points based on summary statistics, user locations, actions, and activity times
- These insights indicate that focusing on premium customers in Italy and the US with mobile app access and cash or credit card payments might yield the highest sales revenues currently.
- October 6, 2020 stands out as the highest day both in sales and returns, indicating a day of very high transaction volume but also significant return activity.
